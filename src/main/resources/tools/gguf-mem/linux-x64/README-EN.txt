GGUF VRAM Calculator Tool Instructions
======================================

If your version is missing the gguf-mem executable, please download and place it manually.

Download URL:
https://github.com/IIIIIllllIIIIIlllll/llama.cpp-vram-calculator/releases/download/releases/release-linux-x64.zip

After extracting, place gguf-mem at the following path:

    classes/tools/gguf-mem/linux-x64/gguf-mem

The final directory structure should be:

    classes/
      tools/
        gguf-mem/
          linux-x64/
            gguf-mem

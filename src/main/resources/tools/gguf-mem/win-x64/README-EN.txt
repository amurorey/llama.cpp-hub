GGUF VRAM Calculator Tool Instructions
======================================

If your version is missing the gguf-mem executable, please download and place it manually.

Download URL:
https://github.com/IIIIIllllIIIIIlllll/llama.cpp-vram-calculator/releases/download/releases/release-win-x64.zip

After extracting, place gguf-mem.exe at the following path:

    classes\tools\gguf-mem\win-x64\gguf-mem.exe

The final directory structure should be:

    classes\
      tools\
        gguf-mem\
          win-x64\
            gguf-mem.exe
